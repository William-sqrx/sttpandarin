Pinyin sound map

How every pinyin syllable is broken into a starting sound (initial) + a rest (final), each matched to a sound inside a familiar English or Bahasa word — e.g. hǎo → “h” like hat + “ao” like now. This is the same table the in-app pronunciation coach card uses. Edit any word or the underlined sound below; changes save in this browser and update the live preview. Use Copy JS / Download JSON to take edits back to the app.

Tone marks or numbers both work (e.g. hao3).

Initials

The starting consonant of a syllable (the longest match wins, so zh/ch/sh beat z/c/s).

Finals

Everything after the initial (the vowel/rime). Keyed by the spoken rime — pinyin spelling like ju, yao, zhi is normalized to these keys before lookup.

Whole-syllable overrides

The buzzed -i syllables (zi/ci/si/zhi/chi/shi/ri) have no separable vowel, so the whole syllable maps as one unit.

Split finals (reference)

Rimes with no single familiar word are taught as two chunks — a glide + a base rime that each reuse a Final above. Read-only.
